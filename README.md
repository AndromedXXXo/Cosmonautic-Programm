# Космонавтика

Курсовая работа по дисциплине "Объектно-ориентированное программирование".

## Сборка и запуск

```
./gradlew run
```

или

```
./gradlew build
java -jar build/libs/cosmonautics-1.0.0.jar
```

## Структура

- `model/` — классы предметной области (иерархия космических объектов)
- `service/` — сервисный слой (репозиторий, поиск, статистика, хранилище)
- `ui/` — графический интерфейс (Swing)
